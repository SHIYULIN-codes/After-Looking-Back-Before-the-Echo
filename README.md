# After Looking Back, Before the Echo

An interactive text system focused on reconstructive memory and the boundaries of reflection.

It is an attempt to understand the past, while also generating new versions of it; in this process, thought grows echoes, and eventually, a way out.


## Technologies Used

- **p5.js**  
  Used for canvas rendering, animation, visual layout, text generation, and interactive drawing.

- **ml5.js FaceMesh**  
  Used to detect facial landmarks through the camera.

- **Camera Input**  
  Used to read:
  head position
  head rotation
  body distance
  eye-closed state
